ICONOS PARA EXTENSION DE ARCHIVOS

Esta página muestra un ejemplo de cómo utilizar los íconos personalizados según la extensión de archivo.
El desarrollo se basa en Bootstrap 5.3 y JavaScript, y lo más importante está organizado en las siguientes carpetas:

fonts/cpa-files
Contiene los archivos de fuente que definen los íconos como tipografía.
css/cpa-files
Incluye el archivo de estilos con las clases para aplicar los íconos, estilos :before, colores y tooltips personalizados.
demo.html
Archivo de ejemplo donde se demuestra cómo aplicar los íconos en una interfaz web.
Importante: este archivo contiene codigo HTML y estilos CSS para fines visuales, no es necesario incluirlos en tu proyecto.

Usar como Boton o Link

BOTON

<button type="button" class="btn btn-cpa-file" 
  data-bs-toggle="tooltip"
  data-bs-placement="top"
  data-bs-custom-class="edr-tooltip"
  data-bs-title="Descargar archivo PDF"
>
  <i class="cpa-file cpa-file-pdf"></i>
</button>

LINK (A.HREF)

<a href="#" class="btn btn-cpa-file"
  data-bs-toggle="tooltip"
  data-bs-placement="top"
  data-bs-custom-class="edr-tooltip"
  data-bs-title="Descargar archivo XLS"
>
  <i class="cpa-file cpa-file-xls"></i>
</a>

Recomendaciones para el uso de iconos en tablas

Para alinear varios iconos horizontalmente dentro de una celda, se recomienda envolverlos en un 

    <div class="d-flex justify-content-around">
      <!-- aquí van los iconos -->
    </div>
Esto asegura que los iconos se distribuyan uniformemente y mantengan una presentación ordenada.

Si deseas usar una tabla con filas rayadas (estilo "striped"), se sugiere alternar los colores de fondo entre #F4F5FA y #FFFFFF para mejorar la legibilidad.

Si estás trabajando con Bootstrap 5, puedes aplicar este estilo personalizado agregando la siguiente regla CSS: 

    .table{ --bs-table-striped-bg:  #F4F5FA !important; }
Esto sobrescribirá el color por defecto que Bootstrap usa para las filas alternadas.

Activación de tooltips en Bootstrap 5.3 con JS

Para que los tooltips funcionen correctamente, es necesario inicializarlos con JavaScript. Bootstrap no los activa automáticamente, así que debes agregar el siguiente código al final de tu documento, idealmente antes de cerrar la etiqueta </body> :

  <script>
    const tooltipTriggerList = document.querySelectorAll('[data-bs-toggle="tooltip"]');
    const tooltipList = [...tooltipTriggerList].map(tooltipTriggerEl => new bootstrap.Tooltip(tooltipTriggerEl));
  </script>

Este script busca todos los elementos que tengan el atributo data-bs-toggle="tooltip" y les aplica la funcionalidad del tooltip. Sin este código, los tooltips no se mostrarán aunque tengas los atributos correctamente definidos.
